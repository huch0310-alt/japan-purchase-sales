# 多语言功能设计规格

## 1. 项目概述

- **项目**: 日本采销管理系统 - 多语言支持
- **目标**: 添加日/中/英三语言切换功能
- **用户**: 系统管理员和员工

## 2. 技术方案

### 技术选型
- **vue-i18n**: Vue 3 官方推荐的国际化方案
- **Pinia**: 状态管理，与现有项目一致
- **localStorage**: 语言偏好持久化

### 依赖
```json
{
  "vue-i18n": "^9.x"
}
```

## 3. 功能设计

### 3.1 语言切换
- 位置: 头部导航栏右侧
- 方式: 下拉菜单
- 语言: 中文(简体)、日本語、English
- 持久化: localStorage 保存偏好

### 3.2 翻译范围
仅导航和菜单文字：
- 侧边栏菜单项
- 路由 meta title
- 头部用户信息

### 3.3 翻译 Key 约定
```javascript
// 示例
nav.dashboard: 信息统计
nav.customer: 客户管理
nav.staff: 员工管理
nav.product: 商品管理
nav.category: 分类管理
nav.order: 订单管理
nav.invoice: 账单管理
nav.report: 报表统计
nav.settings: 系统设置
nav.logs: 操作日志

header.logout: 退出登录
```

## 4. 文件结构

```
web-admin/src/
├── locales/
│   ├── zh.js      # 中文简体
│   ├── ja.js      # 日语
│   └── en.js      # 英语
├── store/
│   └── language.js # 语言状态管理
└── main.js        # 注册 i18n
```

## 5. 实现步骤

1. 安装 vue-i18n 依赖
2. 创建语言文件 (zh.js, ja.js, en.js)
3. 创建语言状态管理 (Pinia store)
4. 在 main.js 中注册 i18n
5. 修改 Layout.vue 添加语言切换按钮
6. 修改路由配置使用翻译 key
7. 修改各页面使用翻译 key
